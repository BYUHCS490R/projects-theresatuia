Explain the difference between inline, embedded, and external CSS?

Inline CSS is coded straight into HTML, Embedded is CSS typed ino the header using style tags, External is calling a file in your repository to execute CSS properties in your HTML.

Explain why you should only use external CSS?

if you use embedded or inlined it'll override style sheets forwebdesigners. It also makes finding a unified design or changes to those design more simple.

What is a selector used for?

Selectors are used to target certain elements of your HTML to change their style, by identifying and targeting tags
