ASCII and Unicode

Both are code that allow representation of characters. ASCII allows for display of english characters, letters, symbols, and letters using a 7-bit numerical code. Unicode is more modern, it assigns uique code to every character in english and other languages.

What is UTF-8

UTF stands for Unicode Tranformation Format, used to represent the Unicode's characters. A sequence of 1 to 4 bytes per character that repest entire Unicode characters.

URL

URL stands for Uniform Resource Locator and is the address to webpages and resource located on the internet.

Domain Name

Used to replace complex IP addresses with simple and easy to read addresses to access resources on the internet such as "https://"

HTTP

a component of a URL that stands for Hypertext Transfer Protocol and sets rules and protocols between web browsers and servers

HTTPS

A secure version of http and enccrypts data between browser and websites for privacy and sensitive infomation, this applies to logins and payments done on the web

IP Address

Stands for Internet Protocol Address, allows the transfering of data between devices by recognizing a numerical label assigned to devices on a network that supports IP addresses

DNS

Stands for Domain Name System, converts website names into IP Addresses that computers need to locate eachother.

W3C Recommendations

Standards and specifications for technology that are endorsed by the World Wide Web Consortium certifies and publishes.
